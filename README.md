# clj-mastery
A collection of domain-specific Clojure applications and practical drills. Covers areas such as Big Data (via Spark), AI/ML, and Computational Math. Notably, this repository demonstrates seamless Java ecosystem integration, incorporating necessary Java interop code directly to solve real-world engineering challenges.
